Open Database Connectivity (ODBC) ()
Name: Open Database Connectivity (ODBC) Description: A programming language-independent programming interface to databases originally derived from CLI. It has much in common with it, and is more widely used. ODBC can also provide connectivity to some data sets that do not support SQL, including spreadsheets. Source/URL: Refer to: https://docs.microsoft.com/en-us/sql/odbc/reference/odbc-programmer-s-reference.
