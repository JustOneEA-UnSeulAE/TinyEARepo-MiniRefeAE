Logging ()
Logging helps provide insight to the current and past activities taking place within the services. By default, Edge components use a logging level of INFO. However, you can set the logging level for each Edge component. For example, you might want to set it to DEBUG for the Message Processor, or to ERROR for the Management Server. the associated message level helps increase or decrease the verboseness of the outputted information to help debug a situation that may be taking place, or monitor ongoing activities of a solution to catch any errors as they occur
