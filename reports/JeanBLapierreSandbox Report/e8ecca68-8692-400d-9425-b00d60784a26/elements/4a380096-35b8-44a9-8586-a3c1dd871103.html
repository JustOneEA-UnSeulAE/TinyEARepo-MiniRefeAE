Apache ZooKeeper ()
Apache ZooKeeper is a software project of the Apache Software Foundation. It is essentially a service for distributed systems offering a hierarchical key-value store, which is used to provide a distributed configuration service, synchronization service, and naming registry for large distributed systems. https://en.wikipedia.org/wiki/Apache_ZooKeeper
