Guideline A.1.1: (For the authoring tool user interface) Ensure that web-based functionality is accessible. [Implementing A.1.1] http://www.w3.org/TR/2015/NOTE-IMPLEMENTING-ATAG20-20150924/#gl_a11 Rationale: When authoring tools (or parts of authoring tools) are web-based, conforming to WCAG 2.0 (https://www.w3.org/TR/ATAG20/#conf-rel-wcag) will facilitate access by all authors, including those using assistive technologies. A.1.1.1 Web-Based Accessible (WCAG): If the authoring tool contains web-based user interfaces, then those web-based user interfaces meet the WCAG 2.0 success criteria. (Level A to meet WCAG 2.0 Level A success criteria; Level AA to meet WCAG 2.0 Level A and AA success criteria; Level AAA to meet all WCAG 2.0 success criteria) Implementing A.1.1.1 http://www.w3.org/TR/2015/NOTE-IMPLEMENTING-ATAG20-20150924/#sc_a111
