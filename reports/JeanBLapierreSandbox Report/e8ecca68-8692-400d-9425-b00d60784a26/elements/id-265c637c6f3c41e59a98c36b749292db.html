Data Fabric and Data Mesh ()
Data Fabric and Data Mesh A data fabric is an architected system that provides uniform access to data held in multiple, disparate sources. To do this, it uses data integration as the underlying implementation. A data mesh is an architected system which, like a data fabric, uses data integration implicitly, but which need not provide uniform access, and is based on data products. Relatively few enterprises have data fabrics or data meshes today, but their use is growing.
