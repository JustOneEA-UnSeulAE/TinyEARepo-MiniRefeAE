Consul ()
"Consul is a service networking solution to connect and secure services across any runtime platform and public or private cloud. Features (Consul lingo): * Mesh Gateway (https://www.consul.io/mesh.html). Connect between different cloud regions, VPCs and between overlay and underlay networks without complex network tunnels and NAT. * Layer 7 Traffic Management (https://www.consul.io/docs/connect/l7-traffic-management.html) * Consul includes its own built-in L4 proxy (https://www.consul.io/docs/connect/proxies.html)"
