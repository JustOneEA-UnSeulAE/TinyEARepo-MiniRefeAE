A7.03.15 Development Languages [DACM] ()
A7.03.15 Development Languages A development (programming) language is an artificial language used to write instructions that can be translated into machine language and then executed by a computer. This also includes Administration scripting languages (e.g. Unix Shell Scripting, Power Shell, etc.), as well as Query Languages (PL/SQL, T-SQL).  Markup Languages (HTML, XML, etc.) are located in a different standard. A7.03.15 Langages de développement Un langage de développement (programmation) est un langage artificiel utilisé pour rédiger des instructions qui peuvent être traduites en langage machine, puis exécutées par un ordinateur.    Cette norme comprend également les langages de script d’administration (p. ex., Unix Shell Scripting, Power Shell, etc.), ainsi que les langages de requête (PL/SQL, T-SQL).  Les langages de balisage (HTML, XML, etc.) sont abordés dans une norme différente.
