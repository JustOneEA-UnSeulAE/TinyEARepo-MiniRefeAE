OAG Audit of Cyber Security of Personal Information in the Cloud - KEY FINDINGS ()
KEY FINDINGS Overall, the audit found that: There were weaknesses in departments’ controls for preventing, detecting, and responding to cloud cyberattacks. The roles and responsibilities for ensuring cloud cybersecurity were unclear. TBS did not provide departments with a costing model or funding approach for cloud serves. PSPC and SSC did not include environmental criteria in their procurement of cloud services. The OAG report did not state that personal data stored in the cloud has been compromised or that security breaches had occurred.
