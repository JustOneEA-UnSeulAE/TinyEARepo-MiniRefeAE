P2 - Data Stewardship and Accountability Are Mandatory Description Every enterprise dataset must have clearly defined accountability, decision rights, and stewardship, aligned with departmental governance structures. Rationale The Data Accountability Framework and Governance Operating Model establish that clear accountability is foundational to effective, trusted data use and risk management. Impacts • Every significant dataset requires an identified Data Steward. • Architecture and solution designs must align with defined data accountabilities. • Unowned or “orphan” datasets are not acceptable in enterprise solutions. • Steward responsibilities include data quality, access decisions, and documentation.
