P1 - Data Is an Enterprise Asset Description Data created, collected, or managed by ESDC is a shared enterprise asset and not owned by a single program, system, or branch. Data must be managed for departmental value, reuse, and long-term stewardship. Rationale The Strategy emphasizes that better outcomes for Canadians depend on employees treating data as shared, protected enterprise assets, supported by stewardship and collaboration across silos. Impacts • Solutions must design data for reuse beyond a single application or program. • Data duplication and siloed ownership models are discouraged. • Enterprise data platforms (e.g., Data Lake, Data Warehouse, Catalogue) become preferred integration points. • Governance bodies may override local optimization to preserve enterprise value.
