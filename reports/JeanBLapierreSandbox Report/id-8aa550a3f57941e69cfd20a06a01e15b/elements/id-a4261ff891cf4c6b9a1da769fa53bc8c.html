A4.03.06 Data Quality Assurance [DACM] ()
A4.03.06 Data Quality Assurance Software and or services that support data quality assurance. Data quality assurance is the process of profiling the data to discover inconsistencies and other anomalies in the data, as well as performing data cleansing activities (e.g. removing outliers, missing data interpolation) to improve the data quality. These activities can be undertaken as part of data warehousing or as part of the database administration of an existing piece of applications software. Was: A4.04.04 A4.03.06 Assurance de la qualité des données Logiciel ou service permettant d’assurer la qualité des données. L’assurance de la qualité des données désigne le processus de profilage des données visant à découvrir les incohérences et d’autres anomalies dans les données, ainsi que la tenue d’activités d’épuration des données (p. ex., élimination des valeurs aberrantes, interpolation des données manquantes) visant à améliorer la qualité des données.  Ces activités peuvent être entreprises dans le cadre de l’entreposage des données ou de l’administration des bases de données d’un logiciel d’application existant.
