A1.08.07 Data Lake [DACM] ()
A1.08.07 Data Lake A data lake is a storage repository that holds a vast amount of raw data in its native format until it is needed. While a hierarchical data warehouse stores data in files or folders, a data lake uses a flat architecture to store data. A1.08.07 Lac de données Un lac de données est un dépôt de stockage qui renferme une grande quantité de données brutes dans leur format d’origine jusqu’à ce que quelqu’un en ait besoin. Un entrepôt de données hiérarchique stocke des données dans des fichiers ou des dossiers, alors qu’un lac de données utilise une architecture plate pour le stockage de données.
