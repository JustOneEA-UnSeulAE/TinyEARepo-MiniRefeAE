Working Together To Support IT Security ()
Working together to support IT security. Far more than the sum of the tools used to protect information and systems, an effective IT security program combines people, processes and technologies. Each department's senior managers, program and service delivery managers, security personnel, IT operational personnel, human resources personnel and other stakeholders work together in a concerted manner to achieve the same high level of IT security across the federal government.
