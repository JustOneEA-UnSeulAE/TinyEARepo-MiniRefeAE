Conventions of Data Architecture ()
2 Conventions of Data Architecture The EDRM is constructed in accordance with both principles and conventions. Conventions provide guidance on arbitrary choices of form that should help stewards and data architects define the EDRM with minimum arbitrary decisions. Note that the word conventions as used here means invented, agreed upon standards.
