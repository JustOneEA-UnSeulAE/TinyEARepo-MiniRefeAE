A8.02.03 Extract Transform and Load (ETL) [DACM] ()
A8.02.03 Extract Transform and Load (ETL) Software that supports the extraction of data from a data source, the manipulation and change of data to a different format and the population of data destination with the transformed data. Narrower Terms: Data Mapping A8.02.03 Extraction, transformation et chargement (ETC) Logiciel qui prend en charge l’extraction de données d’une source de données, la manipulation et le changement des données vers un format différent, et l’ajout des données transformées à l’emplacement de destination.    Terme spécifique : mise en correspondance des données.
