Data Platforms Data platforms are architecture components that connect to multiple sources and provide access to their data. They are key components of data fabrics and data meshes. They may import data from the sources but more usually, and more effectively, they use data virtualization. In either case, they transform the source data to fit a target schema, or expose the source schema to applications. Figure 3 shows how data platforms enable applications to access data.
