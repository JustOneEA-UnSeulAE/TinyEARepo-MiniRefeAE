A8.02 Data Interoperability [DACM] ()
A8.02 Data Interoperability Data Interoperability defines the software and or services for sharing data and services across disparate systems and vendors. Related Terms: Enterprise Application Integration (EAI), Enterprise Service Bus (ESB), Middleware, API Management (Application Programming Interface). A8.02 Interopérabilité des données L’interopérabilité des données définit les logiciels ou les services de partage de données, ainsi que les services entre différents systèmes et des fournisseurs disparates.    Termes connexes : intégration des applications d’entreprise (EAE), bus de services d’entreprise (BSE), intergiciel.
