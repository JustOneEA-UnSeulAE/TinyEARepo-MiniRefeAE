A1.08.08 Web Analytics Web analytics is the measurement, collection, analysis and reporting of internet data for purposes of understanding and optimizing web usage. There are two categories of web analytics; off-site and on-site web analytics. A1.08.08 Analytique Web L’analytique Web permet d’assurer la mesure, la collecte et l’analyse des données Internet afin de comprendre et d’optimiser l’utilisation du Web et de produire des rapports. Il existe deux catégories d’analytique Web : l’analytique Web hors site et sur place.
