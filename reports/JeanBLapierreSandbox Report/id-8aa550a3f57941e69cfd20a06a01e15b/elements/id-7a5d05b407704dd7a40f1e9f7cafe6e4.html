A4.08.05 Geospatial Data Analysis [DACM] ()
A4.08.05 Geospatial Data Analysis Supports the application of statistical analysis and other informational techniques to geographically based data. Also known as Geostatistics. A4.08.05 Analyse de données géospatiales Prend en charge l’application de techniques d’analyse statistique et d’autres techniques d’information aux données géographiques. Aussi connu sous le nom de géostatistique.
