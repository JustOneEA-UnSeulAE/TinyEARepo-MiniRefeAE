Foundational Data Infrastructure (Business Service) ()
Foundational Data Infrastructure All of the work across ESDC to turn data into knowledge and insights that inform our decision-making is made possible by the underlying Foundational Data Infrastructure that allows us to collect, store, transform, integrate, and securely share our data assets to where they need to be across the department. Foundational Data Infrastructure helps us meet the organisation’s need for data access by using up-to-date platforms, software, and tools to collect, organise, store, manage, and use our collective data.
