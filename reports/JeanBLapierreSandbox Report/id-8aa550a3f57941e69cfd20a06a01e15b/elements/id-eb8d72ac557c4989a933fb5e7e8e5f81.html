A8.02.02 Data Transformation [DACM] ()
A8.02.02 Data Transformation Data transformation is the process of converting data or information from one format to another, usually from the format of a source system into the required format of a new destination system.  Source: Techopedia Related Terms: Data Structure Dictionary, Extract Transform Load (ETL), Data Mediation, Mediation System Narrower Terms: Data Mapping A8.02.02 Transformation des données La transformation des données consiste à convertir des données ou des renseignements d’un format à un autre, habituellement du format d’un système source au format requis d’un nouveau système de destination.  Termes connexes : dictionnaire de la structure des données, extraction, transformation et chargement (ETC), médiation des données, système de médiation.  Terme spécifique : mise en correspondance des données.
