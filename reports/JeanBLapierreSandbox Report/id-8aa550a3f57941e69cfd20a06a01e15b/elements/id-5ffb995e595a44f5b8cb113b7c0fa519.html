Citizen Service Branch (CSB) (Stakeholder) ()
Citizen Service Branch (CSB) Client Experience (CX) within CSB has been engaged from the early phase of program design. CX brings a client centric perspective to help program designers understand pain points a client may experience as they seek information / apply for the program / seek support in requesting program / application related information and help to ensure a barrier free experience. In-Person Directorate within CSB provides live in person support to persons enquiring about the program and/or to persons requesting assistance applying for the program and/or enquiring about the status of an application.
